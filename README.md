# DACA portfoolio — Sirja Hass

## Minust
Liitusin DACA programmiga, et arendada oma andmeanalüüsi ja andmeinseneeria oskusi ning saada praktilist kogemust SQL-i, visualiseerimise ja Pythoniga. Siia portfooliosse koondan iga nädala individuaalsed tööd, meeskonna väljundid, peamised leiud ja refleksioonid.

## Eesmärk
Õppida andmeid uurima, puhastama, ühendama, visualiseerima ja automatiseerima ning ehitada töödest selge ja professionaalne portfoolio.
