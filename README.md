# DACA portfoolio — Sirja Hass

## Minust

Arendan DACA programmis praktilisi andmeanalüüsi ja andmeinseneeria oskusi. Portfoolio koondab minu SQL-i, Power BI, Pythoni, API-de ja andmetöötluse projektid koos äriliste järelduste ning meeskonnatöö väljunditega.

## Põhioskused

- **SQL:** andmete uurimine, puhastamine, JOIN-id, agregatsioonid, CTE-d ja aknafunktsioonid
- **Visualiseerimine:** Power BI, KPI-d, dashboard'i disain ja andmelood
- **Python:** pandas, Plotly ja RFM-kliendisegmenteerimine
- **Andmetöötlus:** Supabase API, ETL-pipeline, veakäsitlus ja automatiseerimine

## Valitud projektid

1. [Turunduse Power BI dashboard](week-5/README.md) — kanalite tulemuslikkus, kliendihankimise trend ja juhtimisvaade
2. [RFM-kliendisegmenteerimine Pythonis](week-7/README.md) — kliendisegmendid, visualiseeringud ja turundussoovitused
3. [API-põhine ETL-pipeline](week-8/README.md) — automaatne andmete pärimine, töötlemine, visualiseerimine ja eksport
